# [Kabuki Toolkit Cookbook](../readme.md)

## KabukiPress

KabukiPress is an extendable set of tools for writing, editing, and publishing blogs, books, and documents to various file formats and statically generated websites using IMUL and Markdown. This chapter will teach you how to create a book, a blog for the book, and an online store with tracked analytics.
